# 🔥 2025-2026 AI Agent 热门项目全景报告

> 📅 更新时间：2026-05-15 | 📊 数据来源：GitHub API 实时采集

---

## 📌 六大关键趋势

1. **从框架到平台**：Dify、n8n、Mastra 等项目将 Agent 能力封装为可视化平台，降低使用门槛
2. **MCP 协议一统天下**：Anthropic 的 Model Context Protocol 已成为 Agent-工具通信的事实标准
3. **终端 Agent 爆发**：Claude Code、OpenAI Codex CLI、Aider 等终端编码 Agent 竞争激烈
4. **多 Agent 协作标配**：AutoGen、CrewAI、MetaGPT、OpenAI Agents SDK、Google ADK 全部支持多 Agent 编排
5. **记忆与状态成关键**：mem0、Letta 等项目专注于解决 Agent 的记忆和持久化问题
6. **Web/手机 Agent 兴起**：Browser-use、Agent-S、Open-AutoGLM 让 Agent 能直接操作网页和手机
